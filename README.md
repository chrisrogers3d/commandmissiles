Inspired by the original Missile Command game and @Xulipa's Wand Duel and @Aidan's Dreampark.

Wanted to reinterpert this game as a mixed reality, immersive version.  I think the game mechanic works in WebXr.

now that we are seeing people getting attacked with ballistic missiles regularly its different now.  my sympathy to people living in fear or injured because of these weapons.

my first webxr project, most of my vr experience is unity with a little unreal.  i really liked the iteration time of this webxr, vite hotreloaded it much quicker than a unity build.

there needs to be a (fake) cert for you to serve https locally.  claude did that for me.

the ar/mr version works on phones, i made this for the quest3.
